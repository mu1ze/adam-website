🎵 SPOTIFY

Music playback control and playlist management

Home / Plugins / Spotify

// Overview

Control Spotify playback through ADAM. Play music, search for tracks, manage playlists, and get information about what's currently playing — all through voice or text commands.

ACTIVE Connected via Spotify Web API

// Capabilities

  • Play, pause, skip tracks
  • Search for songs, albums, artists
  • Manage playlists (create, add, remove)
  • Control volume and shuffle
  • View currently playing
  • Add to queue
  • Get track recommendations

// Example Usage

> play "Bohemian Rhapsody" > what's currently playing > skip to next song > add this to my "Workout" playlist > search for "chill lofi beats" > set volume to 50% > pause the music

// Configuration

Provider: Spotify Web API Auth: OAuth 2.0 Device: Active device Playlists: Linked to account
