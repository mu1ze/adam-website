// Overview

Access and manage your Obsidian vault. ADAM can read notes, create new content, search through your knowledge graph, and maintain your personal wiki — all through natural language.

ACTIVE Vault: Adam-vault

// Capabilities

  • Read and search all notes in vault
  • Create and edit markdown files
  • Navigate backlinks and wikilinks
  • Create daily notes and templates
  • Manage vault structure and folders
  • Extract and summarize note content
  • Sync notes to GitHub

// Example Usage

> search my vault for "project ideas" > read my daily note from yesterday > create note "Meeting Notes 2026-03-26" > find all notes linking to this one > summarize the project reference folder > add "TODO" section to current note
