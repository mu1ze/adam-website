🗂️ OBSIDIAN

Navigate your knowledge graph, create notes, and connect ideas

Home / Plugins / Obsidian

// Overview

The Obsidian plugin syncs with your knowledge base. Manage notes, link ideas, and retrieve information from your personal wiki directly within ADAM.

ACTIVE

// Capabilities

  • Read and write notes
  • Create links between notes
  • Search your vault
  • Organize notes into folders
  • Sync with remote Obsidian Git repositories

// Example Usage

> obsidian: read note "Project Ideas" > obsidian: create note "Meeting Summary 2026-03-26" > obsidian: link "AI Research" to "New Skill" > obsidian: search "graph neural networks"
