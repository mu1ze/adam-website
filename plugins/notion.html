// Overview

Manage your Notion workspace through ADAM. Create and edit pages, manage databases, update properties, and automate your Notion workflow with natural language.

ACTIVE Connected via Notion API

// Capabilities

  • Create and edit pages
  • Query and update databases
  • Add and update properties
  • Create child pages and blocks
  • Search workspace content
  • Archive and restore pages
  • Manage comments and discussions

// Example Usage

> create page "Project Update" in workspace > add new row to "Tasks" database > update task status to "Done" where name = "Test" > search for notes about "quarterly report" > get all items in my inbox database > archive this completed page

// Configuration

Provider: Notion API v2 Auth: Internal Integration Token Database: Multiple workspaces Blocks: All supported types
