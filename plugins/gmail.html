// Overview

Manage your Gmail account with natural language. ADAM can read your inbox, compose and send emails, search through messages, and help you stay on top of important communications.

ACTIVE Connected via Gmail API

// Capabilities

  • List and read emails from inbox
  • Compose and send new emails
  • Reply to existing threads
  • Search emails by sender, subject, or content
  • Mark emails as read/unread
  • Archive and delete emails
  • Manage labels and filters

// Example Usage

> show my unread emails > send email to john@example.com about the meeting > reply to latest email from HR > search for emails from last week about "project" > mark this email as important > create label "ADAM" for automated emails

// Configuration

Provider: Gmail API v1 Auth: OAuth 2.0 Account: Configured via OpenClaw Labels: INBOX, SENT, DRAFT, SPAM
