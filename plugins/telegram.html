Send messages, manage groups, and receive notifications

Home / Plugins / Telegram

// Overview

The Telegram plugin enables seamless communication via Telegram. Send messages to contacts or groups, manage chats, and get real-time notifications directly.

ACTIVE

// Capabilities

  • Send messages to users and groups
  • Create and manage channels
  • Get notifications
  • Manage chat members
  • Use bots for automated interactions

// Example Usage

> telegram: send "Hello!" to "@username" > telegram: send "Meeting Reminder" to group "Project Team" > telegram: notify channel "Updates" about new release
