// Overview

Send and receive Telegram messages through ADAM. Perfect for getting notifications, sending alerts, or managing group chats — all through natural language commands.

ACTIVE Connected via Bot API

// Capabilities

  • Send messages to any chat
  • Receive and process incoming messages
  • Manage group chats
  • Send notifications and alerts
  • Handle callbacks and inline queries
  • Send media (photos, files, voice)
  • Manage bot commands and keyboard

// Example Usage

> send message "Meeting in 5 minutes" to @username > list recent messages from group > send photo to channel > create poll "What time?" with options > reply to last message with "Got it!"

// Configuration

Provider: Telegram Bot API Auth: Bot Token Chat ID: Configured Commands: /start, /help, /status
