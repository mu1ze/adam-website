🎨 IMAGE GEN

Generate stunning images with DALL-E, Flux, and more

Home / Plugins / ImageGen

// Overview

The ImageGen plugin creates images from text prompts using various AI models. Generate artwork, illustrations, and visuals for any purpose.

ACTIVE

// Capabilities

  • Generate images from text descriptions
  • Support for multiple AI models (DALL-E, Flux, Stable Diffusion)
  • Adjust image resolution and aspect ratio
  • Generate variations of existing images
  • Create transparent PNGs

// Example Usage

> imagegen: "A futuristic cityscape at sunset" --model DALL-E > imagegen: "A cat wearing a wizard hat" --resolution 512x512 > imagegen: "Impressionist painting of a sunflower" --style vibrant
