// Overview

Intelligent routing that selects the best LLM for each task while optimizing costs. ClawRouter routes requests across 55+ models from OpenAI, Anthropic, Google, DeepSeek, and more.

ACTIVE v0.12.81

// Capabilities

  • Automatic model selection based on task
  • Cost optimization across providers
  • 55+ models from 15+ providers
  • 11 free models always available
  • Latency and throughput optimization
  • Provider failover and redundancy
  • Usage analytics and reporting

// Available Models

FREE TIER (11 models): - Nemotron Ultra 253B (reasoning) - DeepSeek V3.2 (coding) - Mistral Large 675B (multilingual) - GPT-OSS 120B (general) + 7 more... PREMIUM: - GPT-4, Claude, Gemini Pro - DeepSeek, Grok, and 45+ more
