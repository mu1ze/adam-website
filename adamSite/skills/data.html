// Overview

The Data module processes, analyzes, and visualizes data. ADAM can work with CSVs, JSON, databases, and APIs to extract insights, generate reports, and create charts.

ACTIVE

// Capabilities

  • Parse and analyze CSV/JSON files
  • SQL query generation and optimization
  • Statistical analysis and trends
  • Data cleaning and transformation
  • Report generation from datasets
  • API data aggregation

// Example Usage

> analyze: this sales CSV and find trends > query: get top 10 customers by revenue > clean: remove duplicates from this dataset > chart: create bar chart of monthly data > report: summarize key metrics from JSON
