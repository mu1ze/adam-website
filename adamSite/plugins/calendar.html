📅 CALENDAR

Schedule events, set reminders, and manage your time

Home / Plugins / Calendar

// Overview

The Calendar plugin integrates with your schedule. ADAM can create events, set reminders, and check your availability, keeping you on track with your day.

ACTIVE

// Capabilities

  • View upcoming events
  • Create new events with details
  • Set reminders for events
  • Check availability
  • Reschedule or cancel events
  • Sync with Google Calendar

// Example Usage

> calendar: show me my schedule for tomorrow > calendar: create event "Team Meeting" for Friday 2pm > calendar: remind me to call John at 10am > calendar: check availability for next week
